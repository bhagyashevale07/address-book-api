# Address Book API

## Features
- Create Address
- Get Address
- Update Address
- Delete Address
- Nearby Search API

## Technologies Used
- FastAPI
- SQLite
- SQLAlchemy
- Pydantic

## Installation

```bash
pip install fastapi uvicorn sqlalchemy pydantic geopy
```

## Run Project

```bash
uvicorn main:app --reload
```
